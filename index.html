Mac Install Instructions

==For a Clean Mac install==

install brew: http://brew.sh/

== Python 2.7 ==

brew install python

== PostGresQL ==

brew install postgres

== Pip==

sudo easy_install pip

Check pip is installed: pip

Should return: Usage: pip COMMAND [OPTIONS] pip: error: You must give a command (use "pip help" to see a list of commands)

== VirtualEnv ==

install virtualenv

sudo pip install virtualenv

== Git ==

Following this link: http://git-scm.com/download/mac

==PyCharm==

https://www.jetbrains.com/pycharm/download/

==Requirements Setup==

Go to requirements page: https://github.com/anandology/broadgauge

== Set up PostGreSQL ==

First we need to fix an issue with PostGres and Windows.

  1. Open Start menu, type "Command"
  2. Find a "Command Prompt" option, right-click it and select "Run as Administrator"
  3. Run
    notepad "C:\Program Files\PostgreSQL\9.4\pg_env.bat"
  4. Edit and replace
    @SET PATH="C:/Program Files/PostgreSQL/9.4\bin";%PATH%
    with
    @SET PATH=C:/Program Files/PostgreSQL/9.4\bin;%PATH%
    (i.e. just remove the double quotes) so it should look like:

@ECHO OFF REM The script sets environment variables helpful for PostgreSQL

@SET PATH=C:\Program Files\PostgreSQL\9.4\bin;%PATH% @SET PGDATA=C:\Program Files\PostgreSQL\9.4\data @SET PGDATABASE=postgres @SET PGUSER=postgres @SET PGPORT=5432 @SET PGLOCALEDIR=C:\Program Files\PostgreSQL\9.4\share\locale

  1. Hit save

==Setting Up Virtual Env and Requirements==

In PyCharm, open the "requirements.txt" file. Comment out the "psycopg2" line, by putting a # in front of it. This module is broken in Windows. Instead need to install it manually:

http://www.stickpeople.com/projects/python/win-psycopg/ Download the "Python 4.7" version that suits your laptop (32bit or 64bit)

Open a Terminal in PyCharm(hover mouse over bottom left icon, select Terminal) Run these commands:

virtualenv . Scripts\activate

You are now inside a Virtual Environment, whose only purpose is to serve this website.
To set up the environment in Pip, run the following command: "C:\Program Files\PostgreSQL\9.4\pg_env.bat"

Now install the win-psycopg2 download with: easy_install C:\Users\FAKE\FOLDERPATH\psycopg2-2.6.0.win-amd64-py2.7-pg9.4.1-release.exe

Next: pip install -r requirements.txt

Create a database: createdb pythonexpress

Add schema: psql pythonexpress < broadgauge/schema.sql

Run the app: python run.py

In your web browser open:
http://localhost:8080

==NOTE==

You don't have to create a new env everytime. Just run the following command: Scripts\activate

Run the app: python run.py
